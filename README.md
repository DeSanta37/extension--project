## MedLens: Описание продукта

### Суть концепта
**MedLens** — браузерное расширение для студентов-медиков, которое превращает сложные академические и клинические тексты в понятный материал, адаптированный под уровень подготовки пользователя. Работает прямо на странице, не требует копирования и сохраняет исходную структуру документа.

---

### MVP

| Функция                    | Описание                                                                                                           |
| :------------------------- | :----------------------------------------------------------------------------------------------------------------- |
| **Адаптация по выделению** | Выделите мышью абзац или фрагмент > расширение автоматически предложит упрощение.                                  |
| **Умный Popup**            | Панель настроек открывается по клику на иконку в тулбаре или через контекстную подсказку над выделенным текстом.   |
| **Замена на странице**     | Оригинальный текст мгновенно заменяется на адаптированную версию. Вёрстка, ссылки, списки и заголовки сохраняются. |
| **Мгновенный возврат**     | Плавающая кнопка «Оригинал» позволяет переключаться между версиями в один клик без перезагрузки.                 |
| **Перевод EN > RU**        | Встроенный режим перевода с английского с учётом медицинской терминологии и клинического контекста.                |
| **Экспорт в PDF**          | Опциональная функция сохранения адаптированного фрагмента в PDF для офлайн-чтения или печати.                      |

---

### Почему именно **MedLens**?
Название объединяет **Medical** + **Lens** (линза).  
Как оптическая линза фокусирует свет, отсекая лишнее, так и MedLens фокусирует внимание на сути медицинского текста, делая сложное понятным без потери научной точности.

## Работа с Git

cd medlens-extension
npm install

### 2. Начало задачи
git checkout main
git pull origin main
git checkout -b feature/(Название ветки)

### 3. Работа
git add .
git commit -m "feat: (Описание исправлений)"
