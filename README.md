
# БРАУЗЕРНОЕ РАСШИРЕНИЕ ДЛЯ АДАПТАЦИИ СЛОЖНОСТИ ТЕКСТА
<br>
<br>

> [!important]
> ## О ПРОЕКТЕ

Мы создаём кроссплатформенное браузерное расширение, которое работает в Chrome и Firefox на базе WebExtensions API. Пользователь заходит на любой сайт, видит сложный текст (научная статья, юридический документ, техническая документация), выделяет его мышкой или нажимает кнопку «Вся страница», выбирает один из трёх уровней упрощения, и расширение заменяет оригинальный текст на адаптированную версию, сгенерированную нейросетью GigaChat. После этого пользователь может в любой момент вернуться к оригиналу одним кликом.
